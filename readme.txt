Easily enable modern security headers for your website with the Strict Security Headers plugin, with no configuration required.

== Description ==

Strict Security Headers is a straightforward and lightweight plugin designed to enhance the security of your WordPress website by implementing modern security headers. Simply activate the plugin, and the headers are automatically added, there's absolutely no configuration needed!

== Features ==

1. **Simple and Lightweight:** No configuration needed, just activate the plugin to enhance your website security.
2. **Enhanced Security:** Helps in protecting your website against various types of attacks and vulnerabilities.
3. **No Performance Impact:** Strict Security Headers is developed to have minimal impact on your website's performance.

== Installation ==

1. Upload the `strict-security-headers` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. That's it! Your website now has enhanced security headers. You can verify it by checking on [Security Headers](https://securityheaders.com).

== Frequently Asked Questions ==

= Is there any configuration required? =
No, there's absolutely no configuration required. Just activate the plugin and you're good to go.

= How can I check if the security headers are properly set? =
You can verify the security headers by visiting [Security Headers](https://securityheaders.com) and inputting your website URL to check the headers applied to your website.
